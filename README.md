## вљЎ Deploy to Vercel (One Click)

1. Click the button above **or** run:

```bash
npx vercel --prod
```

2. Set these **Environment Variables** in your Vercel dashboard:

| Variable | Required | Description |
|---|---|---|
| `DNS_SERVERS` | No | Comma-separated DNS servers (default: `8.8.8.8,1.1.1.1`) |
| `MT_HOST` | Optional | MikroTik router IP (for API push) |
| `MT_USER` | Optional | MikroTik username |
| `MT_PASS` | Optional | MikroTik password |
| `MT_PORT` | Optional | MikroTik API port (default: `8728`) |

3. Done! Your app is live.

## Example MikroTik Script Output

```rsc
# Generated by MikroTik Blocker
# Date: 2026-03-28T17:00:00.000Z
# Domains: facebook.com, tiktok.com
# List: blocked_sites

# Step 1: Remove existing entries for these domains (clean update)
/ip firewall address-list remove [find list=blocked_sites comment="facebook.com"]
/ip firewall address-list remove [find list=blocked_sites comment="tiktok.com"]

# Step 2: Add resolved IPs to address list
/ip firewall address-list
add list=blocked_sites address=157.240.241.35 comment="facebook.com"
add list=blocked_sites address=157.240.221.35 comment="facebook.com"
add list=blocked_sites address=161.69.137.89 comment="tiktok.com"

# Step 3: Add firewall drop rule (idempotent)
/ip firewall filter
:if ([:len [find chain=forward dst-address-list=blocked_sites action=drop]] = 0) do={
  add chain=forward dst-address-list=blocked_sites action=drop comment="Block blocked_sites" place-before=0
}
```
