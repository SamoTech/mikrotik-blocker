## Quick Start

### Prerequisites
- Node.js >= 18
- npm or yarn

### Backend

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Environment Variables

See `backend/.env.example` for all options.

| Variable | Description | Default |

| `DNS_SERVERS` | Comma-separated DNS servers | `8.8.8.8,1.1.1.1` |

## Example MikroTik Script Output

```rsc
# Generated by MikroTik Blocker
# Date: 2026-03-28T17:00:00.000Z
# Domains: facebook.com, tiktok.com

/ip firewall address-list
add list=blocked_sites address=157.240.241.35 comment="facebook.com"
add list=blocked_sites address=157.240.221.35 comment="facebook.com"
add list=blocked_sites address=161.69.137.89 comment="tiktok.com"
add list=blocked_sites address=161.69.137.90 comment="tiktok.com"

/ip firewall filter
add chain=forward dst-address-list=blocked_sites action=drop comment="Block Sites" place-before=0
```
